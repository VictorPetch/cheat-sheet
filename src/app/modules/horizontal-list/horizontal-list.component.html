since divs have display:block, they tend to occupy the same width of the parent.

This div has display: flex. That means divs inside it won't have the same width of the parent.

You can set width to 100% so that both divs occupy as much space as possible

However, let's say we have a dynamic number of divs. They can easily overflow the parent div. You can use the overflow-x: auto to achieve that

If we just set the width property, the divs are flattened to don't cause overflow.

Even though their width is 200px, on the screen they are less than that.

To achieve overflow, we'll use min-width: 200
