Text overflow

Text Overflow can be done with css: 

			text-overflow: ellipsis;
			overflow: hidden;
			white-space: nowrap;
asdasdsad asd asd asd asd asdasd asdasdaasd asd adsa sdasd asd

However, that only works for 1 line and the element that has the text needs to have a set width or max-width.
If you want to have a div with responsive width that has a variable text that can be too big, you can use this: 

			text-overflow: ellipsis;
			overflow: hidden;
			white-space: nowrap;
			display: -webkit-box;
			-webkit-line-clamp: 1;
			-webkit-box-orient: vertical;
asdasdsad asd asd asd asd asdasd asdasdaasd asd adsa sdasd asd

Change -webkit-line-clamp value to set the max number of lines before the overflow is forced.
Note that the element that contains the text has a responsive width (usually 100% to make it use all the remaining space). It won't work if you do it like this: 

			
				asdasdsad asd asd asd asd asdasd asdasdaasd asd adsa sdasd asd
